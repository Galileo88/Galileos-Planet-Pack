## GPP_Rescale

GPP_Rescale is an optional mod that rescales the GPP solar system to either 2.5x or 10x its normal size. Rescaling is accomplished without the use of Sigma Dimensions. GPP_Rescale works with any of the following installations:

  * GPP + GPP_Rescale
  * GPP + GPP_Rescale + GEP + GEP_Rescale
  * GEP + GEP_Primary + GEP_Rescale + GPP + GPP_Secondary + GPP_Rescale
  * JNSQ + GPP + GPP_Secondary + GPP_Rescale
  * JNSQ + GPP + GPP_Secondary + GPP_Rescale + GEP + GEP_Rescale

For any other combination of planet packs, rescaling should be performed using Sigma Dimensions without GPP_Rescale.

To install GPP_Rescale:

1. Download and install Galileo's Planet Pack per the installation instructions.

2. Dowloand and install any other applicable planet packs and optional mods (see options above).

3. Drill down to [GPP Download]\Optional Mods\GPP_Rescale\ and select either Rescale_2.5X or Rescale_10X.

4. Copy the selected GameData folder into your KSP install.
